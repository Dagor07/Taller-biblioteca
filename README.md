# Taller-biblioteca ***MENU***

## **Opción 1**
### - Selecciona la opcion 1 para agregar cualquier tipo de libro, excepto si esta repetido.

## *Opción 2*
### - Selecciona la opcion 2 para ver todos los libros que hay en la biblioteca, incluyendo lo que estan prestados.

## *Opción 3*
### - Selecciona la opcion 3 para buscar un libro. Puedes encontrarlo por Autor, titulo y/o categoria.

## *Opción 4*
### - Selecciona la opcion 4 para poder llevarte un libro y devolverlo cuando lo termines. (Maximo de libros 3 por persona. Gracias.)

## *Opcion 5*
### - Selecciona la opcion 5 para devolver a la biblioteca un libro prestado.

## *Opción 6*
### - Selecciona la opcion 6 para ver cuantos y cuales libros tienes prestados.

## *Opción 7*
### - Selecciona la opcion 7 para eliminar un libro de la biblioteca permanentemente.

## *Opción 8*
### - Selecciona la opcion 8 para salirte del programa (BIBLIOTECA)
